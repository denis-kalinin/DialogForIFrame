Wizard in the Dialog

This page is in the iframe. In case you want to close a parent dialog window from the iframe, that iframe MUST send inter-window message to the parent window: window.parent.postMessage('close-dialog', '*'). You can use another message instead of close-dialog but set that name in the constructor of DialogForIFrame.

We don't close iframe via parent window, because they can be in different domains, so window.parent.frames[0].close() won't work in a such case.
